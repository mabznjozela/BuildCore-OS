Run and deploy your AI Studio app
This repository contains everything you need to run your app locally.

Run Locally
Prerequisites: Node.js

Install dependencies:
Run npm install in your terminal.

Set your API Key:
Create a file named .env.local in the main folder and add your Gemini API key inside it like this:

Plaintext
GEMINI_API_KEY=your_actual_api_key_here
Run the app:
Run npm run dev in your terminal, then open your browser and go to http://localhost:3000 to view the app.

Once you paste and save this version, your repository is completely clean, safe, and ready to share!
